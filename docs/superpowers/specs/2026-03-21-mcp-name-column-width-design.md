# MCP 列宽自适应收窄设计

**日期：** 2026-03-21

**目标**
- 让 `MCP` 页面中的“名称 / 传输方式 / 启用状态”列按内容自适应收窄。
- 为这些列设置最大宽度，避免它们占据过多表格空间。
- 对超长内容使用省略号，并保留完整悬浮提示。

**方案**
- 保留原生 `table` 布局，不改动现有交互和数据结构。
- 给前三列增加 `width: 1%`，向浏览器传达“尽量按内容收窄”的意图。
- 在名称文本容器上设置 `max-width` + `text-overflow: ellipsis`。
- 在 `传输方式`、`启用状态` 的 `pill` 容器上设置最大宽度，并让内部文字支持单行省略。
- 使用 `title` 暴露完整文本。

**影响范围**
- `app-v2/src/pages/ServersPage.tsx`
- `app-v2/src/styles/ui.css`

**不做**
- 不改 `skill管理` 页面列表。
- 不改表格为 `grid` / `flex`。
- 不调整其他列的交互或语义。
